# Project Overview

***Event Buddy is an event management application designed to streamline the organization and participation in events.
It
offers a user-friendly interface for both event organizers and attendees, providing features such as event creation,
registration and booking.***

It is a (solo-coded) learning project focused on modern backend technologies, including Spring Boot and MongoDB. The
backend API is well-documented in Swagger and includes comprehensive testing using JUnit 5 and Docker Testcontainers to
ensure reliability and performance.

For the Frontend the project is built on React with TypeScript, ShadCn Components and Orval-Client.

### Main Features

- **Organization Management**: Users can create and manage organizations hosting events.
- **Event Creation**: Organizers can create and manage events with detailed information.
- **Bookings**: Attendees can register for events and manage their bookings.
- **Dashboard**: A comprehensive dashboard for organizers to monitor event statistics and attendee information.

#### Side Features

- User Authentication (OAuth2) and role/ownership-based Authorization mechanisms
- User Profile and (visibility) Settings Management
- Image upload for Organizations/Events

### Upcoming Features

- Notification/Mailing/Message System (Email/In-App)
- Guest List Download (PDF)
- Improved Queries on Overview Pages (Pagination, Filtering, Sorting)
- Personalized Front-Page for Users
- Authentication improvements with Magic-Links and additional Providers
- Multiple Image support on Events/Organizations in Frontend and Backend
